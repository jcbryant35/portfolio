Memory Game

This is a Mardi Gras themed Memory Game. It was built because I wanted show that my skills are not only limited to websites.

This game was built to be responsive (as all of my websites are) so that users can enjoy the same experience on any device. Whether it is a phone, tablet, laptop, or desktop PC, it is guaranteed to deliver a great deal of fun.

It was built with plain JavaScript and all of the cards on the game board are created and loaded dynamically. This was a very fun project!
