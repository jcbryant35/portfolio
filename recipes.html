Down South Recipes App

Down South Recipes is a full stack application packed full of flavor with delicious southern recipes.

It was built with the MERN stack. (Mongo, Express, React, Node). This was my first full stack application published on the internet.

This web application features lots of recipes by category, and a rating and reviews feature for each recipe. Also features a search bar for each category to find specific recipes without scrolling through them all.

What's next?


This web application will soon undergo a design change, and several new features will be added.
