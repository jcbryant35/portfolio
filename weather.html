Weather App

This a very simple weather application that has the ability to show the current weather forecast and also the five day weather forecast. You can search by city name or via current GPS location.

This website utilizes the Fetch API, and an external third party API called 'Open Weather Map' to successfully retrieve accurate weather data and display it to the user.
